ルール説明
  • 6×6 盤の標準リバーシ。黒が先手です。
  • 初期盤面（中央4石）以降に打たれた石にはマークが付きます（黒石は 🐱、白石は 🐶）。反転されるとマークは消えます。
  • 打った石と自分のマーク石で相手の石を挟んで返すとコンボ成立。1手の獲得ポイントはポイント式 f(R, C, D) で決まります（R = その手で返した枚数、C = コンボ数 = マーク石で挟んだライン数、D = 確定コンボなら1）。式はプリセット（殴りデッキ / その他）から選ぶか、カスタムで自由に書けます。式を変えると過去の手も含めて全ポイントが再計算されます。
  • 確定コンボ（D=1）: 打った石が相手の次の1手では絶対に返されず、かつ相手がどう応じても次の自分の手番でその石を起点にコンボできる場合です。次のコンボが約束された手といえます。D はその手で打った石だけを判定します。
  • 自分のマーク石を相手に反転されるとコンボ元が消えるので、マーク石を守りながら挟むのがポイントです。
  • 両プレイヤーは初期HP（デフォルト各 30000、黒白別に変更可）を持ち、獲得ポイントはそのまま相手への与ダメージになります。相手のHPを先に0にした方の勝ちです。盤面が埋まる／両者連続パスの場合は残りHPが多い方の勝ちです。
  • 打てる場所がない場合は自動でパスします（与ダメ0）。
  • 黄色のマスは次に打てる場所、橙色のマスはその手でコンボが成立する場所です。各マスには獲得ポイント（右上の「+○○」）と、その手の R / C / D の値（下部）を表示します。青枠は解析による推奨手です。
  • 「最善プレイ解析」は、両者が最善を尽くした場合の最終HP差（KOによる勝敗確定を含む）を αβ探索で読んだ値です。読みの地平はゲーム開始から16手目に固定です（読む深さ = 16 − 経過手数）。読み切るまで計算し、途中経過は「暫定」として随時表示されます。KO時もオーバーキル分を含む最終HP差を表示します。
